## About The Project

This project is a static portfolio website designed to be fast, responsive, and easy to maintain. It features a clean and modern design to highlight my work and expertise in various web technologies.

Key features include:
- A responsive layout that works on all devices.
- A project section to showcase my recent work.
- An "About Me" section with my skills and experience.
- A contact form for easy communication.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js (v18 or higher)
- npm

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/apondi-art/portfolio.git
   ```
2. Navigate to the project directory
    ```sh
    cd portfolio
    ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Start the development server
    ```sh
    npm run dev
    ```
   Open [http://localhost:4321](http://localhost:4321) to view it in the browser.

## Deployment

This project is automatically deployed to GitHub Pages whenever there is a push to the `main` branch. The deployment workflow is defined in the `.github/workflows/deploy.yml` file.
